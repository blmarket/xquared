Full Architecture

This document is for developers that want to develop expansion features for Xquared (autocompletion, autocorrection, templates, redefined keyboard shortcuts, etc.) or for developers that want to edit our code.

 

Xquared's main modules and module relationships 

xq.Editor

xq.Editor class handles

  • the conversion between editing mode and standard mode
  • event processing between keyboard shortcuts, mouse clicks, etc.
  • all editing command processing
  • All settings (autocompletion, autocorrection, etc.) processing


xq.rdom.Base

xq.rdom.Base class provides

  • default CRUB of browser DOM objects
  • high level DOM constructive API in inferring the difference between browsers
  • various query methods to obtain specified DOM nodes
  • method for caret and selection administration

 

xq.Validator

xq.Validator class executes

  • conversion of HTML to appropriate form in editing
  • Whitelist based HTML Validation
