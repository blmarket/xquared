이 문서는 Xquared의 확장 기능(자동 완성, 자동 변환, 템플릿 처리기, 단축키 재정의 등)을 개발하거나, 코드를 수정하고자 하는 개발자를 위해 작성되었습니다.

 

Xquared의 주요 모듈 및 모듈 간의 관계

xq.Xquared

 

xq.Editor

xq.Editor 클래스는

  • 편집모드/일반모드 사이의 전환
  • 키보드 단축키, 마우스 클릭 등의 이벤트 처리
  • 각종 편집 명령 처리
  • 각종 설정(자동 완성, 자동 변환 등) 처리

등을 담당합니다.

 

xq.rdom.Base

xq.rdom.Base 클래스는

  • 브라우저 DOM 객체에 대한 기본적인 CRUD
  • 브라우저 간의 차이를 추상화한 고수준의 DOM 조작 API
  • 특정 DOM 노드를 얻어내기 위한 다양한 쿼리 메서드
  • 캐럿, 선택(selection) 관리를 위한 메서드

등을 제공합니다.

 

xq.validator.Base

xq.validator.Base 클래스는

  • 편집에 적합한 형태로 HTML을 변환
  • Whitelist 기반 HTML Validation

을 수행합니다.
