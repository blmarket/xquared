Xquared는 Extensible XHTML Editor의 줄임말로, 이름 그대로 다음과 같은 특징을 같습니다:

 

다양한 확장점 제공

Xquared는 다음과 같은 확장점들을 통해 다양한 용도에 맞도록 쉽게 확장될 수 있습니다:

  • 사용자 정의 단축키
  • 자동 변환(autocorrection) 기능 추가
  • 자동 완성(autocompletion) 기능 추가
  • 사용자 정의 컨텍스트 메뉴
  • 사용자 정의 템플릿 처리기
  • 확장 가능한 Validator
  • 매크로 - 유튜브 동영상, Google Gadget 등을 문서의 원하는 위치에 넣을 수 있습니다.
  • 에디틀릿 - 준비중

 

기능 확장에 대한 자세한 내용은 튜토리얼 및 기능 확장하기를 참고하세요.

 

올바른 XHTML 문서를 생성

올바른 XHTML 문서라고 하면 흔히 다음 두 가지만을 떠올리지만:

  • Well-formed
  • Valid

Xquared는 이 두 가지에 더하여 세 번째 원칙을 추가적으로 준수하는 편집기 입니다:

  • Meaningful

예를 들어, 들여쓰기를 하기 위한 용도로 blockquote 태그를 사용하는 것은 "well-formed"이면서 "valid"하기도 하지만 의미적으로는 올바르지 않습니다. 대부분의 웹 기반 편집기는 이와 같은 부적절한 마크업을 생성하는데, 이는 접근성이나 호환성 등을 낮추는 주된 원인 중 하나입니다. Xquared는 의미적으로 올바른 마크업을 생성해줍니다.
