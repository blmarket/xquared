Features

Xquared is short for an extensible XHTML Editor and as implied in this name we have special characteristics that are shown below.



Provision of various expansion points

 

Through the following list of extension points, Xquared can be applied to various purposes for easy expansion.


  • User defined keyboard shortcuts

  • Addition of an auto-correction feature

  • Addition of an auto-complete feature

  • User defined context menu

  • User defined template processor

  • Expandable validator

  • Macro – You can put YouTube, videos, Google Gadgets, etc anywhere on your page.

  • Editlet – being prepared.


For more information on expansion, please refer to our tutorial and feature expansion page.

 

 

Generate Valid XHTML Documents


When we talk about valid XHTML, we usually are talking about two things:


  • Well-formed

  • Valid


Along with these two, Xquared is an editor where the third principle conforms with the other two.


  • Meaningful


For example, indentation is used with the blockquote tag making it well-formed and valid, but it is not meaningful. The majority of web-based editors generate inappropriate markups, thus this is the main cause of lowering accessibility and compatibility. However, Xquared can generate well-formed, valid, and meaningful markups.
